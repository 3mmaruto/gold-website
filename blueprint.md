# Project Blueprint

## Overview
- Marketing website for HVAC/repair services showcasing offerings, company information, and contact options.
- Built with static HTML, Bootstrap-based styling, and supporting CSS/JS assets.

## Implementation Outline
- **Navigation & Hero:** Responsive navbar linking to primary pages, followed by a carousel hero highlighting core services with imagery and calls-to-action.
- **Content Pages:** Individual HTML pages (about, services, contact, team, etc.) provide detailed information using consistent layout and styling.
- **Assets & Styling:** Shared CSS (`style.css`, modular `css/` and `scss/` folders) and JavaScript (`main.js`, `js/`) control presentation, animation, and interactivity. Image assets live under `img/`.
